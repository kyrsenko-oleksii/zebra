Print From Cloud Examples

Here are some examples of using SendFileToPrinter API, based on Zebra Data Services cloud platform. To print from cloud with SendFileToPrinter API, we need to have an API key and have the printer(s) registered with Zebra Data Services. Please apply for an API Key and then register your printer with Zebra Data Services by following the instructions below.

  • How to Apply for an API Key
  • How to Register a Printer with Zebra Data Services
