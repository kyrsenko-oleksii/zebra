How to Use the Example Files

There are a few example files that come with this JavaScript example in this exampleFiles folder. The description about the use of these files is provided below.
  • HelloWorld.txt, could be used for the Select a ZPL File to Send example above. It prints a one-line label.
  • StoreTestFile10M.prn, a test file for the Select a ZPL File to Send example above. It will store the TEST_FILE_10M.txt, a 10MB file, onto the E: drive on the selected printer(s). To remove the TEST_FILE_10M.txt file from the E: drive, send the DeleteTestFile10M.SGD to the printer.
