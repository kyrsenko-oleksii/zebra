How to Use the Example Files

There are a few example files that come with this JavaScript example in this Receipts folder. The description about the use of these files is provided below.
  • SFTPReceipt_ZPL.prn, generated from the SFTPReceipt.nlbl label file by ZebraDesigner for Developer 3, could be used for the Receipt Print example above. It was generated by choosing the "Store template to printer then recall" option and the "Recall stream format" as ZPL. The SFTPReceipt_ZPL.prn file contains both the ZPL Stored Formats and the ZPL Recall Formats of the header, body and footer, a total of six (6) ZPL formats in this file.

  • You could use your own receipt template file for this Receipt Print example, if you have one. Please note that your receipt template file has to be generated by ZebraDesigner for Developer 3 in the same way as described above for SFTPReceipt_ZPL.prn template file
